## Description

**Government School System Website (React Version)** is a **modern, responsive frontend** built using **React + Vite** to represent a government girls‚Äô school system.  
This project aims to bring a **realistic online presence** to schools that lack digital representation featuring home, vision & mission, subjects, and contact page. It provides pages for **Student and Faculty**, while also demonstrating **Local Storage-based data persistence** ‚Äî allowing form data and school information to be saved locally. This React version enhances the previous HTML/Bootstrap project with component-based architecture and reusable UI logic.


## Features

- **React Components** for modular and maintainable code  
- Fully **responsive layout** for all devices  
- **Bootstrap 5 + Custom CSS** for sleek UI  
- **AOS (Animate on Scroll)** for smooth section transitions  
- **Navigation Bar** with active link highlights  
- **Contact Form** with Toast notifications  
- Clean and light UI theme for better readability  
- **Reusable cards and sections** for subjects & vision statements  


## Technologies Used

- **Bootstrap 5**

- **Local Storage API**
- **CSS3 / Flexbox / Grid**
